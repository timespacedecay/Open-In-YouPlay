# Open-In-YouPlay

A userscript to open YouTube links in the YouPlay app on iOS.

Enhanced search engine integration so YouTube links on search result pages (Google, Bing, etc.) open directly in the YouPlay app without loading youtube.com first.

## Downloads
Safari extension to run user script (free): https://apps.apple.com/us/app/userscripts/id1463298887

Userscript code: [Open In YouPlay Userscript][OpenInYouPlay]

## How to install

1. Install the free open source app [Userscripts](https://apps.apple.com/us/app/userscripts/id1463298887)
2. Launch the app and choose path for scripts
3. Enable userscripts on your iPhone in Settings > Safari > Extensions and allow access to sites
4. On iPhone/iPad tap to see the code: [Open In YouPlay Userscript][OpenInYouPlay]
5. Tap Aa in safari and Tap Userscripts
6. Install the detected Userscript.  

## Usage
Open any YouTube link in Safari (either directly or from search results like Google, Bing, DuckDuckGo), and it will automatically redirect to the YouPlay app.

Works with:
- Standard videos (youtube.com/watch?v=...)
- Short links (youtu.be/...)
- Shorts (youtube.com/shorts/...)
- Channels (@username, /c/, /user/)
- Playlists
